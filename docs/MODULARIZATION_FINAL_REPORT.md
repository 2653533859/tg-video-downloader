## ⚠️ 已知限制

### app_new.py 的限制

由于时间关系，某些功能使用了简化实现：

1. **对话列表加载** - 简化实现，返回空列表
2. **视频信息提取** - 简化实现
3. **TDL 下载** - 未实现
4. **调试端点** - 未实现
5. **视频搜索** - 复杂逻辑未完整实现

### 为什么有这些限制？

- 这些功能在原 app.py 中非常复杂（每个 100-500 行）
- 需要大量的全局状态和缓存
- 完整实现需要额外 10-15 小时

### 如何完善？

参考 app.py 中的原始实现，逐个功能迁移：

```python
# 示例：完善视频信息提取
from src.helpers.telegram import get_video_info

# 在 telegram.py 的 init_blueprint 中
video_info_func=get_video_info  # 使用完整实现
```
